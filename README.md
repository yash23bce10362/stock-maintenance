#  Stock Maintenance System

A modern and user-friendly **Stock Maintenance & Inventory Management System** built to help businesses manage products, categories, stock levels, and inventory reports efficiently.  
The interface is clean, responsive, and includes dashboards, item cards, filters, reporting tools, and full CRUD functionality.

---

##  Features Overview

###  1. Dashboard
The dashboard provides a real-time overview of inventory status:

- **Total Items** count  
- **Low Stock Alerts**  
- **Total Inventory Value**  
- **Number of Categories**  
- Quick Action Buttons:
  - Add New Item  
  - View Inventory  
  - Manage Categories  
  - View Reports

---

##  2. Inventory Management
The Inventory page shows all available items in a clean *card-style layout*.

### Key Functionalities:
- View all stock items with:
  - Item Name
  - SKU
  - Description
  - Category
  - Quantity
  - Price
  - Total Value
  - Location
  - Stock Status (e.g., **In Stock**)
- **Search bar** (filter by name, SKU, description)
- **Filter button** for advanced filtering
- **Edit** and **Delete** buttons for each item
- Item counter showing total items at a glance

---

##  3. Add New Item
A full-featured form to add new items into the inventory.

### Fields:
- Item Name  
- SKU  
- Category (select menu)  
- Quantity  
- Price  
- Minimum Stock  
- Maximum Stock  
- Supplier  
- Location  
- Description  

Every field is functional and validates user input.

---

##  4. Categories Management
Manage all product categories used in the system.

### Includes:
- Add new category  
- Edit category  
- Delete category  
- Category display list  
- Category count reflected on dashboard  

---

##  5. Reports
A dedicated reporting module to review inventory analytics.

### Possible actions:
- View inventory history  
- Generate category-wise or item-wise reports  
- View low stock listings  
- Track inventory value changes  
- (Export options depend on implementation)

---

- **Frontend:** React.js / HTML / CSS / Tailwind / Bootstrap  
- **Backend:** Node.js / Express / or any backend configured
